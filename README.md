# seqdia
Sequence diagram editor

![Github Actions](https://github.com/yskszk63/seqdia/workflows/deploy/badge.svg)

## example

```
title Example diagram

Left -> Right: to Right
Right -> Left: to Left
Left --> Right: Dotted
Left ->> Right: Open arrow
```
